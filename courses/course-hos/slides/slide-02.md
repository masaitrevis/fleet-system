# Logbook Requirements

## Required Information

Your logbook must accurately record:

### Driver Information
- Date and total miles driven
- Driver's full name and license number
- Carrier name and address

### Vehicle Information
- Truck number and trailer number
- License plate numbers

### Time Records
- 24-hour period starting time
- Total hours for each duty status:
  - Off-duty
  - Sleeper berth
  - Driving
  - On-duty (not driving)

### Trip Details
- Shipping document numbers
- Pickup and delivery locations
- Driver signature (certifying accuracy)

## Electronic Logs (ELD)

When using ELD:
- Device automatically records driving time
- Driver must still verify and certify records
- Backup paper logs required if ELD fails

## Common Violations

- Falsifying logbook entries
- Missing signatures
- Incomplete duty status changes
- Exceeding daily/weekly limits
